# nutui
这是一款移动端demo，主要使用的技术有vue vueRouter vuex MuseUI NutUI Swiper4等，主要有tab菜单展示，顶部对应页面title展示（返回）、列表展示，轮播，抽屉，card图文展示等等。如果对您有用的话，希望动动您发财的小手，帮我点亮star哦
